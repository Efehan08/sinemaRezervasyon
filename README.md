Description
A simple console-based application written in Java to simulate a cinema ticket booking process. This project was created to practice 2D Arrays, Loops, and Control Flow logic.

Features
Seating Map: Visual representation of a 10x10 cinema hall using a 2D matrix.
Booking System: Users can select and book available seats.
Management: Includes options to cancel or change tickets.
Input Validation: Prevents invalid seat selection.

Technologies
Java
Algorithms (Logic & Flow Control)
